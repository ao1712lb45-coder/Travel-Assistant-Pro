Travel Assistant Pro 1.1（網頁版）

1. 雙擊 START_SERVER.bat。
2. 在「團號」欄位輸入團號，或貼上 Besttour／ITTMS 行程網址。
3. 按「自動抓取並解析」。
4. 確認名稱、價格、日期、航空公司與行程亮點。
5. 產生 LINE／Facebook／Threads 文案，或使用客戶需求配對。

主要功能：
- Besttour／ITTMS 單團自動解析
- 團號、城市、天數、航空與日期辨識
- LINE、Facebook、Threads 文案
- Besttour 行程資料庫同步
- 客戶地區、月份、預算與完整行程內容搜尋

本版已移除圖片 DM 製作與 PNG 下載功能。
