# 社群文案選項

Travel Assistant Pro 提供 LINE、Facebook、Threads 三平台文案。

## 文案風格

- 自動判斷
- 優惠促銷
- 自然分享
- 精簡重點
- 活潑吸睛
- 專業介紹
- 季節旅行
- 放鬆度假
- 親子同行
- 質感行程

## 文案版型

可指定版型 1 至 8，或選擇「每次自動變化」。按下「換一組三平台文案」會同時更新三個平台，並保留各平台適合的格式：

- LINE：精簡、保留網址與聯絡方式。
- Facebook：完整介紹、網址與行動呼籲。
- Threads：自然口吻，不加入網址與聯絡方式。

文案只會使用已解析或手動填寫的行程資料，不會自行加入保證出團、限量席次、不拉車等未確認內容。
