# Travel Assistant Pro 2.0

## 產品方向

V2.0 延續 1.1.0 的官網行程擷取、完整行程搜尋、客戶配對與行程資料庫，不加入需要額外 API 費用的 AI 聊天助理。

## 新增功能

- 全新旅行社工作台視覺，改善桌面與手機操作。
- 一鍵產生六種素材：LINE、Facebook、Threads、EDM、福委提案與 30 秒短影音腳本。
- 客戶 CRM：儲存聯絡方式、喜好、避開條件與備註。
- 行程適配檢查：依目前載入行程比對客戶喜好與衝突條件。
- 所有新增內容以已擷取或人工確認的行程資料為準，不捏造未提供資訊。

## 資料與費用

- 行程與 CRM 資料目前儲存在使用者瀏覽器的 localStorage。
- 不呼叫生成式 AI API，因此日常操作不會產生 AI token 費用。
- 既有 Render 網址與 Node.js 啟動方式保持不變。

## 部署原則

V2.0 先在 `develop/2.0-integrated` 驗證；完整測試通過後，才合併至 `main` 並由 Render 更新既有正式網站。
